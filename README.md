# MT4 HST to CSV converter written in go

Convert .hst file to OHLCV formatted CSV

## Installation

```
go get github.com/miiton/hst2csv
go install github.com/miiton/hst2csv
```

## Usage

```
hst2csv USDJPY.hst
```
